Question-2:What is the difference between innerHTML, innerText, and textContent ?

Ans: 1. By using innerHTML We can get or set html content inside an element.
2.By using innerText we can get or set text inside an element.but ignores hidden elements.
3.By using textContent we can get or set all text content,including hidden elements.

Question-3: What is event delegation in the DOM?

Ans: Event Delegation is a JavaScript pattern that allows us to handle events. Events can be added to a parent element instead of adding to every single element.

Question-4: What is event bubbling in the DOM?

Ans: We can handle multiple child elements using a single event listener on the parent.
